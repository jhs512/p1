## 시리즈 개요

Java 함수 개념을 중학생 대상으로 단계적으로 설명하는 독립 에피소드 시리즈.
각 영상은 독립적이며(이전 강 안 봐도 이해 가능), 하나의 개념만 집중 다룬다.

#### thumbnail

---

#### painScene — 고통
**화면:** println 3줄 순서대로 타이핑
```java
System.out.println("안녕하세요, 민준님!");
System.out.println("안녕하세요, 지아님!");
System.out.println("안녕하세요, 서준님!");
```

**나레이션:**
1. "인사 코드를 이름마다 따로 작성하고 있습니다."
2. "이름만 다를 뿐, 내용은 모두 같습니다."

---

#### conceptScene — 함수 개념
**화면:** 텍스트 중심. "함수" 키워드 강조 + 간단한 도식
(코드 묶음 → 이름 붙이기 → 재사용 화살표)

**나레이션:**
1. "이럴 때 사용하는 것이 함수입니다."
2. "함수는 코드 묶음에 이름을 붙인 것입니다."

---

#### declarationScene — 함수 선언
**화면:** 함수 선언 코드 타이핑
```java
void greet(String name) {
    System.out.println("안녕하세요, " + name + "님!");
}
```
키워드별 색상 하이라이트:
- `void` → 회색 (아직 설명 안 함, 나중에 13강)
- `greet` → 노란색 (함수 이름)
- `String name` → 초록색 (나중에 11강에서 설명)

**나레이션:**
1. "함수는 이렇게 선언합니다."
2. "지금은 함수 이름과 전달받는 값에만 집중하세요."

---

#### callScene — 함수 호출
**화면:** 호출 코드 타이핑 (3줄)
```java
greet("민준");
greet("지아");
greet("서준");
```
화살표 애니메이션으로 함수 선언부 → 호출부 연결 (선택)

**나레이션:**
1. "함수를 사용하려면 이름으로 호출합니다."
2. "같은 함수를 이름 하나로 여러 번 호출할 수 있습니다."

---

#### summaryScene — 요약
**화면:** 요약 카드 2개 + 하단에 코드 미리보기 (기존 에피소드 패턴 동일)
- 카드: `선언은 한 번` / `호출은 여러 번`
- 코드 미리보기 (선택): 선언 + 호출 코드 나란히

**나레이션:**
1. "함수는 반복되는 코드를 이름으로 묶는 방법입니다."
2. "선언은 한 번, 호출은 여러 번 할 수 있습니다."

## 11강 예비 설계 (메모)

**주제:** 매개변수를 쓰면 왜 좋은가
**고통:** `greetMinjun()`, `greetJia()`, `greetSeojun()` — 이름만 다른 함수 3개
**해결:** `greet(String name)` — 매개변수로 하나로 통합
**씬:** thumbnail → painScene → paramScene → callScene → summaryScene

## 12강 예비 설계 (메모)

**주제:** return을 쓰면 왜 좋은가
**고통:** 함수 안에서 println만 함 → 결과를 밖에서 활용 불가
**해결:** return으로 값을 꺼내 변수에 저장, 조건에 활용
**씬:** thumbnail → painScene → returnScene → useScene → summaryScene
